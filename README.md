# AI英语学习平台

基于 Next.js + Supabase + Tailwind CSS 的英语学习平台

## 技术栈
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (Auth + Database)
- shadcn/ui

## 功能特性
- AI 生成个性化学习策略
- 每日任务管理
- 学习进度追踪
- 移动端优先设计

## 开发环境设置

1. 安装依赖：
```bash
npm install
```

2. 配置环境变量：
复制 `.env.local.example` 为 `.env.local` 并填写你的配置

3. 运行开发服务器：
```bash
npm run dev
```

见 `supabase/schema.sql`
